
# Windows Event Log Analysis Dashboard

## Overview

An interactive Power BI dashboard developed to analyze Windows Security Event Logs and visualize authentication-related activities. The dashboard enables users to monitor login events, identify failed authentication attempts, and gain insights into security event patterns through interactive visualizations.

## Features

* Interactive dashboard with dynamic filters
* Login Success vs. Failed Login analysis
* Event ID distribution
* Task Category analysis
* Time-based event trend analysis


## Technologies Used

* Microsoft Power BI
* Microsoft Excel
* Windows Event Viewer

## Dataset

The dashboard is built using Windows Security Event Logs containing authentication events, including successful logins, failed logins, and account management activities.
